# Monotext

**Monotext** is a Rust crate for generating well-formatted, monospaced plain-text documents. It is designed for creating papers, reports, or any structured text content with cover pages, titles, authors, institutions, dates, headers, footers, and automatic page management.

Monotext allows you to focus on the content while ensuring a polished, consistent presentation.

## Features

- Create cover pages with title, subtitle, authors, institutions, and date.
- Automatic page breaks with configurable page height and width.
- Headers and footers with dynamic date, title, and author information.
- Paragraphs, headings, and lists with precise alignment and word wrapping.
- Unicode-aware centering for visually balanced text.
- Roman or Arabic page numbering with optional author abbreviation in footers.
